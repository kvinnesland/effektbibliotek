# CR-005: Materiale og lenker

**Status:** Approved
**Created:** 2026-05-21

## Business Goal
Brukere skal kunne legge til og fjerne lenker (presentasjoner, rapporter, dashboards
m.m.) direkte fra case-detaljsiden.

## Acceptance Criteria
- [ ] Case-detaljside viser eksisterende lenker med type og beskrivelse
- [ ] Caseeier og admin kan legge til ny lenke (tittel, URL, type, beskrivelse)
- [ ] Caseeier og admin kan slette lenker
- [ ] Lenketypene fra FR-MAT-002 er tilgjengelige i dropdown
